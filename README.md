## 🔧 教程
* 需下载python3.11及以下的解释器，python3.12该项目不兼容
* 第一次运行请在主目录执行```pip install -r requirements.txt```命令安装对应的依赖
* 修改config.yaml配置文件
* 当程序可以正常登录和退出后，即可通过login.bat脚本文件在电脑上设置定时任务来实现每天登录网络
### 注：当需要登录多账号的情况下需要以管理员权限运行，否则会出现WIFI切换失败导致登录失败的问题出现
